2. Add **"Check Refine Error"** (Code node) after the Claude call:
   ```javascript
   const response = $input.first().json;
   if (response.error || response.type === 'error') {
     return [{ json: {
       isError: true,
       error_message: response.error?.message || JSON.stringify(response.error || response),
       topic_id: $('Validate').first().json.topic_id,
       project_id: $('Validate').first().json.project_id
     }}];
   }
   return [{ json: { ...response, isError: false } }];
   ```

3. Add **"IF Refine Error"** (IF node): `{{ $json.isError }}` equals true
   - True branch → error path
   - False branch → continue to refine PATCH + Log Refined
