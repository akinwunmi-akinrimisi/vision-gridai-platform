Static image scene clip:
ffmpeg -loop 1 -i {image_path} -i {audio_path} -c:v libx264 -preset fast -crf 23 -pix_fmt yuv420p -c:a aac -b:a 128k -vf "scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2" -r 30 -t {duration_seconds} -movflags +faststart {output_path}

Video scene clip (loop if audio > 5s):
ffmpeg -stream_loop -1 -i {video_path} -i {audio_path} -c:v libx264 -preset fast -crf 23 -pix_fmt yuv420p -c:a aac -b:a 128k -vf "scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2" -r 30 -t {duration_seconds} -movflags +faststart {output_path}

Skipped scene (black + text):
ffmpeg -f lavfi -i color=c=black:s=1920x1080:r=30:d={duration_seconds} -i {audio_path} -vf "drawtext=text='[Scene skipped]':fontcolor=white:fontsize=48:x=(w-text_w)/2:y=(h-text_h)/2" -c:v libx264 -preset fast -crf 23 -pix_fmt yuv420p -c:a aac -b:a 128k -r 30 -t {duration_seconds} -movflags +faststart {output_path}

Concat (no re-encoding):
ffmpeg -f concat -safe 0 -i concat.txt -c copy -movflags +faststart raw_final.mp4

Audio normalization (final only):
ffmpeg -i raw_final.mp4 -af "loudnorm=I=-16:TP=-1.5:LRA=11" -c:v copy -movflags +faststart final.mp4

  <action>
Create a complete n8n workflow JSON for caption generation and video assembly. This is the final production stage, fired by the parallel completion sync from visual workflows.

**Workflow flow:**

1. **Webhook Trigger** — Receives { topic_id } from visual completion sync or manual assembly trigger
2. **Auth Check** — IF node validates DASHBOARD_API_TOKEN
3. **Update Status** — PATCH topics?id=eq.{topic_id}: status='assembling', last_status_change=now()
4. **Load All Scenes** — GET scenes?topic_id=eq.{topic_id}&order=scene_number.asc (includes all timeline data, URLs)
5. **Load Topic** — GET topics?id=eq.{topic_id} (for Drive folder IDs, title)
6. **Log Start** — INSERT production_log: stage='assembly', action='started'

7. **Generate SRT Captions** — Code node (JavaScript):
   - Iterate over scenes with start_time_ms and end_time_ms
   - Skip scenes with skipped=true (no caption for skipped scenes) OR include with "[Scene skipped]" text
   - Format each entry: sequence number, start timestamp (HH:MM:SS,mmm), end timestamp, narration_text
   - Write SRT string to /tmp/production/{topic_id}/captions/{topic_title}.srt

8. **Download Assets** — For each scene that needs a clip built:
   - Download audio file from Drive (or use cached /tmp path if still present)
   - For static_image scenes: download image from Drive/URL
   - For i2v/t2v scenes: download video from Drive/URL
   - Skip scenes with skipped=true (build skipped clip later without downloading)

9. **Build Scene Clips** — Loop over all scenes sequentially (Execute Command nodes):

   For each scene based on type:
   a. **Static image (visual_type='static_image', NOT skipped):**
      ```
      ffmpeg -loop 1 -i /tmp/production/{topic_id}/images/scene_{NNN}.png \
        -i /tmp/production/{topic_id}/audio/scene_{NNN}.mp3 \
        -c:v libx264 -preset fast -crf 23 -pix_fmt yuv420p \
        -c:a aac -b:a 128k \
        -vf "scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2" \
        -r 30 -t {audio_duration_seconds} -movflags +faststart \
        /tmp/production/{topic_id}/clips/scene_{NNN}.mp4
      ```

   b. **Video scene (visual_type IN 'i2v','t2v', NOT skipped):**
      - If audio_duration_ms > 5000 (loop video):
        ```
        ffmpeg -stream_loop -1 -i /tmp/production/{topic_id}/video_clips/scene_{NNN}.mp4 \
          -i /tmp/production/{topic_id}/audio/scene_{NNN}.mp3 \
          -c:v libx264 -preset fast -crf 23 -pix_fmt yuv420p \
          -c:a aac -b:a 128k \
          -vf "scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2" \
          -r 30 -t {audio_duration_seconds} -movflags +faststart \
          /tmp/production/{topic_id}/clips/scene_{NNN}.mp4
        ```
      - If audio_duration_ms <= 5000 (trim video): same command without -stream_loop -1

   c. **Skipped scene (skipped=true):**
      ```
      ffmpeg -f lavfi -i color=c=black:s=1920x1080:r=30:d={audio_duration_seconds} \
        -i /tmp/production/{topic_id}/audio/scene_{NNN}.mp3 \
        -vf "drawtext=text='[Scene skipped]':fontcolor=white:fontsize=48:x=(w-text_w)/2:y=(h-text_h)/2" \
        -c:v libx264 -preset fast -crf 23 -pix_fmt yuv420p \
        -c:a aac -b:a 128k -r 30 -t {audio_duration_seconds} -movflags +faststart \
        /tmp/production/{topic_id}/clips/scene_{NNN}.mp4
      ```

   d. Update scene.clip_status = 'complete' after each clip

10. **Generate Concat List** — Code node or Execute Command:
    ```
    for i in $(seq 1 {scene_count}); do
      printf "file '/tmp/production/{topic_id}/clips/scene_%03d.mp4'\n" $i
    done > /tmp/production/{topic_id}/concat.txt
    ```

11. **Concat** — Execute Command:
    ```
    ffmpeg -f concat -safe 0 -i /tmp/production/{topic_id}/concat.txt \
      -c copy -movflags +faststart \
      /tmp/production/{topic_id}/raw_final.mp4
    ```

12. **Audio Normalization** — Execute Command:
    ```
    ffmpeg -i /tmp/production/{topic_id}/raw_final.mp4 \
      -af "loudnorm=I=-16:TP=-1.5:LRA=11" \
      -c:v copy -movflags +faststart \
      /tmp/production/{topic_id}/final/{sanitized_topic_title}.mp4
    ```

13. **Upload Final Video** — Google Drive Upload to final/ subfolder. Store drive_video_url in topics table.
14. **Upload SRT** — Google Drive Upload to captions/ subfolder.

15. **Update Topic** — PATCH topics: status='assembled', assembly_status='complete', drive_video_url
16. **Log Completion** — INSERT production_log: stage='assembly', action='completed', details: { duration_ms, file_size_mb, scene_count, skipped_count }

17. **Queue Auto-Advance** — GET topics?project_id=eq.{project_id}&status=eq.queued&order=topic_number.asc&limit=1. If a queued topic exists, fire /webhook/production/trigger with that topic_id.

18. **Cleanup** — Execute Command: `rm -rf /tmp/production/{topic_id}/` (only after successful final upload)

Error handling: If FFmpeg fails on any scene clip, mark clip_status='failed' and continue. If concat or normalization fails, set assembly_status='failed' with error details. Do NOT delete temp files on failure.
