**useProjects.js** -- TanStack Query hook:
- `useProjects()`: useQuery with queryKey ['projects'], fetches `supabase.from('projects').select('*')` ordered by created_at desc. Also calls `useRealtimeSubscription('projects', null, [['projects']])` for live updates.
- `useCreateProject()`: useMutation calling `webhookCall('project/create', { niche, description, target_video_count })`. onMutate: optimistic insert with status 'researching'. onError: rollback. onSettled: invalidate ['projects'].

**useNicheProfile.js** -- TanStack Query hook:
- `useNicheProfile(projectId)`: useQuery with queryKey ['niche-profile', projectId], fetches `supabase.from('niche_profiles').select('*').eq('project_id', projectId).single()`. enabled: !!projectId.
- `useProject(projectId)`: useQuery with queryKey ['project', projectId], fetches `supabase.from('projects').select('*').eq('id', projectId).single()`. enabled: !!projectId. Includes Realtime subscription on `projects` table with filter `id=eq.${projectId}`.

**useTopics.js** -- TanStack Query hook:
- `useTopics(projectId)`: useQuery with queryKey ['topics', projectId], fetches `supabase.from('topics').select('*, avatars(*)').eq('project_id', projectId).order('topic_number', { ascending: true })`. enabled: !!projectId. Includes Realtime subscription on `topics` table with filter `project_id=eq.${projectId}`.
- `useApproveTopics(projectId)`: useMutation calling `webhookCall('topics/approve', { topic_ids })`. Optimistic update sets review_status='approved'. onSettled: invalidate ['topics', projectId].
- `useRejectTopics(projectId)`: useMutation calling `webhookCall('topics/reject', { topic_ids, feedback })`. Optimistic update.
- `useRefineTopic(projectId)`: useMutation calling `webhookCall('topics/refine', { topic_id, instructions })`. Sets review_status='refining' optimistically.
- `useEditTopic(projectId)`: useMutation calling `webhookCall('topics/edit', { topic_id, fields })`. Optimistic update with field values.
