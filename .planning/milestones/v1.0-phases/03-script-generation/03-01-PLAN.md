**useScript.js** — Follow useTopics pattern exactly:
- `useScript(topicId)` fetches a single topic with `select('*, avatars(*)')` filtered by `id=eq.topicId`
- Subscribes to Realtime on `topics` table with `id=eq.${topicId}` filter
- Invalidates query key `['script', topicId]`
- Disabled when topicId is falsy

**useScenes.js** — Follow useTopics pattern:
- `useScenes(topicId)` fetches all scenes for topic, ordered by `scene_number` ascending
- Subscribes to Realtime on `scenes` table with `topic_id=eq.${topicId}` filter
- Invalidates query key `['scenes', topicId]`
- Returns empty array when no scenes
- Disabled when topicId is falsy

**useScriptMutations.js** — Follow useApproveTopics/useRejectTopics/useRefineTopic pattern exactly:
- `useGenerateScript(topicId)`: calls `webhookCall('script/generate', { topic_id })`, optimistic update sets status to 'scripting', invalidates `['script', topicId]`
- `useApproveScript(topicId)`: calls `webhookCall('script/approve', { topic_id })`, optimistic update sets `script_review_status` to 'approved', invalidates `['script', topicId]`
- `useRejectScript(topicId)`: calls `webhookCall('script/reject', { topic_id, feedback })`, optimistic update sets `script_review_status` to 'rejected', invalidates `['script', topicId]`
- `useRefineScript(topicId)`: calls `webhookCall('script/refine', { topic_id, instructions })`, optimistic update sets `script_review_status` to 'refining', invalidates `['script', topicId]`
- `useRegenPrompts(topicId)`: calls `webhookCall('script/regen-prompts', { topic_id, scene_ids })`, invalidates `['scenes', topicId]`
- All mutations include `onMutate` (optimistic), `onError` (rollback), `onSettled` (invalidate)

**api.js** — Add 5 helper exports at the bottom of the existing file:
```javascript
export const generateScript = (topicId) => webhookCall('script/generate', { topic_id: topicId });
export const approveScript = (topicId) => webhookCall('script/approve', { topic_id: topicId });
export const rejectScript = (topicId, feedback) => webhookCall('script/reject', { topic_id: topicId, feedback });
export const refineScript = (topicId, instructions) => webhookCall('script/refine', { topic_id: topicId, instructions });
export const regenPrompts = (topicId, sceneIds) => webhookCall('script/regen-prompts', { topic_id: topicId, scene_ids: sceneIds });
```
